Package Installation
---------------------------------

To install the `EGRETci` package you need to be using R 3.0 or greater. Then use the following command:

```R
	install.packages("EGRETci", repos=c("http://owi.usgs.gov/R",
        "http://cran.us.r-project.org"))
```
